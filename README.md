# Homebrew Tap for µTerm

This is the official [Homebrew](https://brew.sh/) tap for [µTerm](https://github.com/ttaatoo/microterm), a lightweight macOS menubar terminal application.

## Installation

```bash
brew tap ttaatoo/microterm
brew install --cask microterm
```

## Upgrade

```bash
brew upgrade --cask microterm
```

## Uninstall

```bash
brew uninstall --cask microterm
brew untap ttaatoo/microterm  # optional
```

## About µTerm

µTerm is a lightweight macOS menubar terminal application built with Tauri 2.0 and Next.js 14.

Features:
- Quick access via menubar icon or global shortcut (Cmd+Shift+T)
- Double-ESC to hide window (vim compatible)
- Customizable opacity and font size
- Multi-tab support

For more information, visit the [µTerm repository](https://github.com/ttaatoo/microterm).
